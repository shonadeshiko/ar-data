## 📏 配置の特徴

- **合計ルート距離**: 450m
- **平均間隔**: 45m（ポスター間）
- **標高範囲**: 2.31m - 3.24m
- **配置パターン**: 元の南船橋データの相対的な位置関係を保持

## 🚀 セットアップ

### 必要なファイル

```
ar-poster-experiment/
├── index.html  (このファイル)
└── poster.jpg  (ポスター画像 - 1枚のみ)
```

### ポスター画像の準備

1. `poster.jpg` という名前でポスター画像を用意
2. 推奨サイズ: 1024 x 1536px（縦長、2:3の比率）
3. ファイルサイズ: 500KB以下推奨（モバイル表示のため）

## 📱 使い方

1. スマートフォンでアクセス（HTTPSが必須）
2. 位置情報とカメラの許可
3. 現地で確認（300m範囲内でポスターが表示されます）

## 📊 技術仕様

- **フレームワーク**: A-Frame 1.4.2 + AR.js 2.2.2
- **位置情報取得**: Geolocation API
- **座標系**: WGS84（GPS標準）
- **表示方式**: 位置情報ベースAR（マーカーレス）

## 📄 付属ファイル

- `gps.csv` - 元の座標データ（南船橋版）
- `gps_relocated.csv` - 移動後の座標データ（文京区版）
